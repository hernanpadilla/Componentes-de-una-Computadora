    🖥️ BIOS - Sistema Básico de Entrada/Salida

    ¿Qué es el BIOS?

    El BIOS (Basic Input/Output System) es un firmware que se ejecuta cuando enciendes tu computadora. Es el primer software que se carga y prepara el sistema para arrancar el sistema operativo.

    Funciones del BIOS:

    • POST (Power-On Self-Test): Verifica que todos los componentes funcionan correctamente
    • Inicialización de hardware: Configura y prepara los componentes básicos
    • Carga del sistema operativo: Localiza y ejecuta el bootloader
    • Configuración del sistema: Permite ajustar parámetros de hardware

    BIOS Tradicional

    • Interfaz de texto básica
    • Limitado a 16 bits
    • Soporte limitado para discos grandes
    • Arranque más lento

    UEFI (Moderno)

    • Interfaz gráfica moderna
    • Soporte para 64 bits
    • Mejor soporte para discos grandes
    • Arranque más rápido y seguro

    Acceso al BIOS/UEFI

    Para acceder al BIOS, presiona F2, F12, DEL o ESC durante el arranque (varía según el fabricante). Una vez dentro, puedes configurar el orden de arranque, overclock, y otras configuraciones avanzadas.

    📐 Factores de Forma de Placas Base

    El factor de forma define el tamaño, forma y distribución de componentes en la placa base. Es crucial para la compatibilidad con el gabinete y otros componentes.

    Factor de Forma Dimensiones (mm) Características Uso Recomendado
    E-ATX 305 × 330 Máxima expansión, múltiples PCIe, soporte para workstations Servidores, workstations de alto rendimiento
    ATX 305 × 244 Estándar popular, buen balance de expansión y tamaño Gaming, uso general, workstations
    Micro-ATX 244 × 244 Más compacto, menos slots de expansión Sistemas económicos, oficina
    Mini-ITX 170 × 170 Muy compacto, un slot PCIe, bajo consumo HTPC, sistemas compactos
    Nano-ITX 120 × 120 Ultracompacto, componentes integrados Sistemas embebidos, IoT

    Consideraciones al Elegir Factor de Forma

    • Compatibilidad del gabinete: Verifica que tu case soporte el factor de forma
    • Necesidades de expansión: Considera cuántos slots PCIe necesitas
    • Espacio disponible: Más grande = mejor ventilación pero requiere más espacio
    • Presupuesto: Las placas más grandes suelen ser más costosas

    ⚙️ Componentes Principales de la Placa Base

    Socket del Procesador

    Conector donde se instala la CPU. Diferentes marcas y generaciones requieren sockets específicos.

    • Intel: LGA 1700, LGA 1200, LGA 2066
    • AMD: AM4, AM5, TR4

    Slots de Memoria RAM

    Ranuras donde se instalan los módulos de memoria. Soportan diferentes tipos y velocidades.

    • DDR4: Estándar actual
    • DDR5: Nueva generación

    Slots de Expansión

    Permiten agregar tarjetas adicionales como GPU, tarjetas de sonido, red, etc.

    • PCIe x16: Para tarjetas gráficas
    • PCIe x8/x4/x1: Para otras tarjetas

    Conectores de Almacenamiento

    Interfaces para conectar discos duros, SSDs y otros dispositivos de almacenamiento.

    • SATA: Para HDD/SSD tradicionales
    • M.2: Para SSDs NVMe de alta velocidad

    Chipset

    Conjunto de chips que controlan la comunicación entre el procesador y los demás componentes. Determina las características y capacidades de la placa base.

    Puertos de E/S (Entrada/Salida)

    Panel trasero que incluye puertos USB, audio, red, video (si hay gráficos integrados), y otros conectores específicos según el modelo.

    🔧 Consejos de Mantenimiento

    Limpieza Regular

    • Usa aire comprimido para remover polvo acumulado
    • Evita líquidos cerca de la placa base
    • Limpia los slots de expansión si no se usan

    Actualizaciones de BIOS

    • Actualiza solo si es necesario (nuevos procesadores, correcciones de bugs)
    • Nunca interrumpas el proceso de actualización
    • Asegúrate de tener energía estable durante la actualización

    Instalación de Componentes

    • Siempre usa pulsera antiestática
    • Verifica compatibilidad antes de instalar
    • No fuerces conexiones - deben entrar suavemente

    Procesador (CPU)

    Guía técnica detallada sobre arquitectura, especificaciones y características del procesador

    🧠 Arquitectura del Procesador

    Componentes Principales

    • Núcleos (Cores): Unidades de procesamiento independientes
    • Cache L1, L2, L3: Memoria de alta velocidad para datos frecuentes
    • Unidad de Control: Coordina la ejecución de instrucciones
    • ALU (Unidad Aritmético-Lógica): Realiza operaciones matemáticas
    • Controlador de Memoria: Gestiona el acceso a la RAM
    • GPU Integrada: Procesamiento gráfico básico (en algunos modelos)

    Especificaciones Técnicas

    Intel

    • Socket: LGA 1700, LGA 1200
    • Arquitectura: Alder Lake, Rocket Lake
    • Proceso: 10nm, 14nm
    • Núcleos: 4-16 (P-cores + E-cores)
    • Threads: 8-24
    • Cache: 16-30MB L3

    AMD

    • Socket: AM4, AM5
    • Arquitectura: Zen 3, Zen 4
    • Proceso: 7nm, 5nm
    • Núcleos: 4-16
    • Threads: 8-32
    • Cache: 16-96MB L3

    Tecnologías Importantes

    • Hyper-Threading/SMT: Múltiples threads por núcleo
    • Turbo Boost: Incremento automático de frecuencia
    • Virtualization: VT-x/AMD-V para máquinas virtuales
    • AES-NI: Aceleración de cifrado

    📊 Rendimiento y Métricas

    Frecuencias

    • Frecuencia Base: Velocidad mínima garantizada
    • Frecuencia Turbo: Velocidad máxima en cargas ligeras
    • Frecuencia de Memoria: Velocidad de acceso a RAM

    Consumo Energético

    • TDP (Thermal Design Power): Potencia térmica de diseño
    • Voltaje: Tensión de operación
    • Estados C: Modos de bajo consumo

    Memoria RAM

    Guía técnica detallada sobre tipos de memoria, especificaciones y configuraciones

    💾 Tipos de Memoria RAM

    Tipo Velocidad Voltaje Pines Características
    DDR4 2133-4800 MHz 1.2V 288 Estándar actual, amplia compatibilidad
    DDR5 4800-7200+ MHz 1.1V 288 Nueva generación, mayor eficiencia
    DDR3 1066-2133 MHz 1.5V 240 Tecnología anterior, aún en uso

    Configuraciones de Memoria

    Canal Simple vs Dual Channel

    • Single Channel: Un módulo, ancho de banda limitado
    • Dual Channel: Dos módulos, doble ancho de banda
    • Quad Channel: Cuatro módulos (workstations)

    Capacidades Comunes

    • 8GB: Mínimo recomendado
    • 16GB: Óptimo para gaming
    • 32GB: Profesional/Creativo
    • 64GB+: Workstations

    ⚡ Especificaciones y Rendimiento

    Timings de Memoria

    • CAS Latency (CL): Retardo en acceso a columna
    • tRCD: RAS to CAS Delay
    • tRP: RAS Precharge Time
    • tRAS: Active to Precharge Delay

    Formato de Timing

    Ejemplo: 16-18-18-38 (CL-tRCD-tRP-tRAS)

    Números más bajos = mejor rendimiento

    XMP/EXPO Profiles

    • XMP (Intel): Perfiles de overclocking automático
    • EXPO (AMD): Equivalent de AMD para DDR5
    • JEDEC: Estándares base de velocidad

    🔧 Instalación y Diagnóstico

    Orden de Instalación

    • Para 2 módulos: Slots 2 y 4 (A2, B2)
    • Para 4 módulos: Todos los slots
    • Siempre consultar manual de motherboard

    Diagnóstico de Problemas

    • MemTest86: Prueba de errores de memoria
    • Prime95: Stress test de estabilidad
    • CPU-Z: Información de especificaciones

    Disco Duro (HDD)

    Guía técnica detallada sobre almacenamiento mecánico y especificaciones

    💽 Tecnología de Discos Duros

    Componentes Principales

    • Platos (Platters): Discos magnéticos donde se almacenan datos
    • Cabezales de Lectura/Escritura: Leen y escriben datos magnéticamente
    • Motor Spindle: Hace girar los platos
    • Actuador: Mueve los cabezales sobre los platos
    • Cache/Buffer: Memoria temporal (8MB-256MB)
    • Controladora: Gestiona operaciones y conexiones

    Especificaciones Técnicas

    Especificación Valores Típicos Impacto en Rendimiento
    RPM 5400, 7200, 10000, 15000 Mayor RPM = mayor velocidad
    Capacidad 500GB - 20TB Más espacio de almacenamiento
    Cache 8MB - 256MB Mayor cache = mejor rendimiento
    Factor de Forma 2.5", 3.5" Tamaño físico del drive

    🔌 Conectividad y Interfaces

    Tipos de Conexión

    SATA (Serial ATA)

    • SATA I: 1.5 Gbps (150 MB/s)
    • SATA II: 3.0 Gbps (300 MB/s)
    • SATA III: 6.0 Gbps (600 MB/s)
    • Conexión de datos + alimentación

    Legacy (Obsoleto)

    • IDE/PATA: 133 MB/s máximo
    • SCSI: Hasta 320 MB/s
    • SAS: Servidores y workstations
    • USB: Discos externos

    📊 Rendimiento y Mantenimiento

    Métricas de Rendimiento

    • Velocidad de Transferencia: 80-200 MB/s típico
    • Tiempo de Acceso: 8-15 ms
    • Seek Time: Tiempo para posicionar cabezal
    • IOPS: Operaciones por segundo (100-200)

    Señales de Falla

    • Ruidos extraños (clicking, grinding)
    • Sectores defectuosos aumentando
    • Tiempos de respuesta lentos
    • Errores de lectura/escritura
    • SMART status crítico

    Herramientas de Diagnóstico

    • CrystalDiskInfo: Monitoreo SMART
    • HD Tune: Test de velocidad y errores
    • SeaTools: Herramienta oficial Seagate
    • WD Data Lifeguard: Herramienta Western Digital

    SSD (Solid State Drive)

    Guía técnica detallada sobre almacenamiento de estado sólido

    ⚡ Tecnología SSD

    Tipos de Memoria NAND

    Tipo Bits/Celda Velocidad Durabilidad Costo
    SLC 1 Muy Alta 100,000 P/E cycles Muy Alto
    MLC 2 Alta 10,000 P/E cycles Alto
    TLC 3 Media 3,000 P/E cycles Medio
    QLC 4 Baja 1,000 P/E cycles Bajo

    Factores de Forma

    2.5" SATA

    • Compatible con HDD tradicional
    • Velocidad: hasta 550 MB/s
    • Conexión SATA III
    • Ideal para upgrades

    M.2 NVMe

    • Conexión PCIe directa
    • Velocidad: hasta 7,000 MB/s
    • Factor de forma compacto
    • Menor latencia

    🚀 Rendimiento y Especificaciones

    Métricas Clave

    • Velocidad de Lectura Secuencial: 500-7,000 MB/s
    • Velocidad de Escritura Secuencial: 450-6,500 MB/s
    • IOPS Random 4K: 50,000-1,000,000
    • Latencia: 0.1-0.2ms (vs 8-15ms HDD)
    • TBW (Terabytes Written): Durabilidad total

    Interfaces de Conexión

    • SATA III: 6 Gbps (600 MB/s límite)
    • PCIe 3.0 x4: 32 Gbps (3,940 MB/s)
    • PCIe 4.0 x4: 64 Gbps (7,880 MB/s)
    • PCIe 5.0 x4: 128 Gbps (15,760 MB/s)

    🔧 Instalación y Optimización

    Configuración en BIOS

    • Habilitar modo AHCI para SATA SSD
    • Verificar detección de M.2 NVMe
    • Configurar como drive de arranque
    • Habilitar CSM si es necesario

    Optimización en Windows

    • TRIM: Automaticamente habilitado
    • Superfetch: Deshabilitado automáticamente
    • Desfragmentación: No necesaria, puede dañar
    • Indexación: Opcional para mejor búsqueda

    🔌 Especificaciones de Potencia

    Cálculo de Potencia Necesaria

    Componente Consumo Típico Consumo Alto Notas
    CPU 65-125W 150-250W Depende del TDP y overclock
    GPU 100-200W 300-450W Componente más demandante
    Motherboard 30-60W 80-120W Incluye slots PCIe y USB
    RAM 3-5W por módulo 8-12W por módulo DDR5 consume más que DDR4
    Storage 2-8W 10-15W SSD consume menos que HDD

    Regla del 80%

    La PSU debe operar máximo al 80% de su capacidad para mayor eficiencia y longevidad.

    Ejemplo: Sistema de 400W necesita PSU de mínimo 500W

    Tarjeta Gráfica (GPU)

    Guía técnica detallada sobre arquitectura, especificaciones y rendimiento

    🎮 Arquitectura y Funciones

    Componentes Principales

    • GPU (Graphics Processing Unit): El "cerebro" que renderiza imágenes
    • VRAM (Video RAM): Memoria de alta velocidad para texturas y datos gráficos
    • Núcleos CUDA/Tensor/RT: Unidades de procesamiento paralelas (NVIDIA)
    • Ray Tracing Cores: Para efectos de iluminación realistas
    • Tensor Cores: Aceleración de IA y DLSS

    Especificaciones Clave

    NVIDIA

    • Series: GeForce RTX 40, 30, 20
    • Memoria: GDDR6, GDDR6X
    • Interfaz: PCIe x16
    • Tecnologías: DLSS, G-Sync, Reflex

    AMD

    • Series: Radeon RX 7000, 6000
    • Memoria: GDDR6
    • Interfaz: PCIe x16
    • Tecnologías: FSR, FreeSync, Smart Access Memory

    💡 Conectores y Requisitos

    Conectores de Poder

    • 6-pin, 8-pin: Conexiones de poder extra de la PSU
    • 12VHPWR: Conector de alta potencia para GPUs modernas

    Salidas de Video

    • HDMI: Estándar para televisores y monitores
    • DisplayPort: Estándar para gaming, soporta altas tasas de refresco
    • USB-C (VirtualLink): Para VR y monitores modernos

    Consideraciones de Compatibilidad

    • Tamaño Físico: Asegura que la GPU quepa en tu gabinete
    • Potencia de la PSU: La GPU es el componente que más consume
    • Monitores: El rendimiento final depende del monitor

    Gabinete (Case)

    Guía técnica detallada sobre tipos de gabinetes, ventilación y gestión de cables

    🌬️ Flujo de Aire y Ventilación

    Configuración de Ventiladores

    Presión Positiva

    Más ventiladores de entrada que de salida. Previene la entrada de polvo, pero puede generar más ruido.

    Presión Negativa

    Más ventiladores de salida que de entrada. Extrae más aire caliente, pero atrae polvo por las rendijas.

    Tipos de Ventiladores

    • PWM (Pulse Width Modulation): Controla la velocidad del ventilador
    • ARGB/RGB: Ventiladores con iluminación
    • High Static Pressure: Para radiadores y disipadores densos

    📐 Factores de Forma de Gabinetes

    Factor de Forma Placas Base Compatibles Características
    Full-Tower E-ATX, ATX, mATX, ITX Máximo espacio, ideal para refrigeración líquida custom
    Mid-Tower ATX, mATX, ITX Más popular, buen balance de tamaño y expansión
    Mini-Tower mATX, ITX Compacto, menos espacio para componentes grandes
    SFF (Small Form Factor) ITX Ultracompacto, especializado para builds pequeños

    🛠️ Gestión de Cables

    Una buena gestión de cables mejora el flujo de aire y la estética del PC.

    • Canales de Enrutamiento: Usar los canales pre-diseñados del gabinete
    • Bridas de Velcro/Zip: Para agrupar cables de forma ordenada
    • PSU Modular: Usar solo los cables necesarios

    Cómo abrir una PC de forma segura

    Apaga completamente la computadora y desconecta el cable de poder
    Presiona el botón de encendido por 10 segundos para descargar la electricidad residual
    Usa una pulsera antiestática o toca el chasis metálico regularmente
    Retira los tornillos del panel lateral (usualmente del lado izquierdo)
    Desliza el panel hacia atrás y levanta para remover
    Toma fotos de las conexiones antes de desconectar cualquier cosa

    Reemplazo de componentes básicos

    Memoria RAM:

    Abre las pestañas de los slots, alinea la muesca de la RAM con el slot, presiona firmemente hasta que las pestañas se cierren automáticamente.

    Disco duro/SSD:

    Desconecta cables SATA y de poder, retira tornillos de montaje, instala nuevo drive y reconecta cables.

    Tarjeta gráfica:

    Desconecta cables de poder PCIe, abre el clip del slot, retira la tarjeta, instala nueva GPU y reconecta todo.

    Fuente de poder:

    Desconecta todos los cables, retira tornillos traseros, instala nueva PSU y reconecta todos los componentes siguiendo las fotos tomadas.
